À propos de TreePestDiag

Objectif

TreePestDiag est né d’un besoin de terrain : centraliser les connaissances dispersées sur les champignons pathogènes des arbres, tout en permettant une recherche inversée par symptômes.

Auteur

Je suis [ton nom ou pseudo], passionné(e) par [la mycologie / la pathologie forestière / l’écologie végétale…]. Mon objectif est de mettre à disposition un outil simple, libre et gratuit pour faciliter le diagnostic des pathogènes arboricoles.

Méthodologie

Les informations proviennent principalement de MycoBank, publications scientifiques et de sites spécialisés. Certaines données ont été initialement collectées à l’aide de ChatGPT, puis revérifiées manuellement.

La base de données est en cours de révision complète pour assurer sa fiabilité. Une colonne "fiabilité" indique le niveau de confiance actuel.

Illustrations

Les illustrations sont intégrées via des liens externes. Aucune image n’est encore hébergée localement, ce qui peut entraîner des disparitions si les sources changent ou suppriment les fichiers.

  • Version optimisée pour les mobiles
